# Build a Survey Form: A Responsive Web Design Project from freeCodeCamp

In this repository, you will find my solution for the "Build a Survey Form" certification project that is part of the Responsive Web Design course at freeCodeCamp.

I was provided with an example page, a link to which can be found in the instructions, which I used to model my own webpage. According to the instructions, my solution only had to be functionally, not visually, similar to the example page.

This is the first webpage I have ever created, and should provide a benchmark for any future improvement.
